# Write BRIEF.md

Create a scoped initiative document for an Archie session to execute.

## Process
1. Identify the gap(s) this initiative closes (reference AEOS criteria by ID)
2. Define objective in one sentence
3. List deliverables with clear definition of done
4. Assign tasks to specific agent roles
5. Estimate CEO time required (batch it)
6. Define risks and mitigations
7. Set duration and timeline

## Inputs
- Gap analysis from `identify-gaps`
- Relevant context from the AEOS document
- Budget remaining

## Outputs
- `BRIEF.md` file ready for Archie to execute

## Quality Criteria
- Every deliverable has a verifiable definition of done
- Every task has a clear owner (agent role)
- The initiative traces to specific AEOS ideal state criteria
- CEO time is batched and minimized
