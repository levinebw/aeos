# Assess Current State

Run a full assessment of the business's current state and update `docs/current-state.md`.

## Process
1. Read the AEOS document for ideal state criteria
2. Check each criterion against available data
3. For each criterion, determine: met / on-track / at-risk / not-started
4. Note any new information that changes the picture
5. Update `docs/current-state.md` with timestamped snapshot
6. Identify the top 3 gaps between ideal and current state

## Inputs
- AEOS document (ideal state criteria)
- Latest available data (analytics, reports, system checks)
- Reports from other agents
- `docs/ledger.md` (budget status)

## Outputs
- Updated `docs/current-state.md`
- Prioritized gap list

## Quality Criteria
- Every ideal state criterion has a status
- No status is based on assumption — must reference data or observation
- Timestamp is current
