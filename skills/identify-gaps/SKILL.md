# Identify Gaps and Prioritize

Analyze the delta between ideal and current state to determine what to work on next.

## Process
1. Read updated current state
2. List all unmet ideal state criteria
3. Score each by: impact (how much does this matter?) × urgency (is there a deadline?) × feasibility (can we close this gap now?)
4. Rank and select the top 1-3 gaps to address
5. For each, determine: which agent(s) are needed? What's the scope? Is this a new initiative or part of an existing one?

## Inputs
- `docs/current-state.md`
- AEOS document (ideal state criteria)
- Active BRIEF.md files (if initiatives are in progress)

## Outputs
- Prioritized gap list with rationale
- Recommendation: new initiative, or adjustment to existing one

## Quality Criteria
- Priorities are justified with reasoning, not just intuition
- Feasibility considers budget, agent availability, and dependencies
